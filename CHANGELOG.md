# Changelog

## 0.5.1

- Version number updates
- Updated README to clarify the current support for the Agents window

## 0.5.0

- Initial release
- Modal confirmation dialog on Enter in all Copilot Chat surfaces (panel, quick chat, inline chat, terminal chat)
- Per-surface configuration to enable/disable confirmation independently
- "Always send for this window" option to suppress the dialog for the current session
- Status bar indicator when confirmation is suppressed
- Commands to re-enable confirmation or disable for the session
